# @oumi/cli

## 安装
```
npm install -g @oumi/cli
# OR
yarn global add @oumi/cli
```

## 使用
```js
oumi ui
```


## 升级
```
npm update -g @oumi/cli
# OR
yarn global upgrade --latest @oumi/cli
```
